### 저장소를 다운로드(클론)하기

깃 사용법을 안다면 이 저장소를 여러분들의 PC로 클론합니다.

> __주의사항__: 
> 이 저장소는 Git LFS를 사용합니다.
> <br>Github Desktop 또는 LFS를 지원하는 각종 git 클라이언트를 통해 클론해주세요.
> <br>깃허브 오른쪽 상단의 초록색 버튼 > 'Downloading Zip'으로는 파일을 정상적으로 다운로드 할 수 없습니다.
